
# Laravel

Project tugas membuat website katalog toko online


## Installation

clone project
```bash
  git clone https://github.com/ahmadsaipullah/MSIB-Eduwork-05.git
```
composer
```bash
  composer install
```
running
```bash
  php artisan serve
```
